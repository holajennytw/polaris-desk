# 研究助理 & 同業比較 — 資料缺口分析

> 追蹤路徑：`api.py` → `retriever.py` → `bigquery_store.py` → BQ `polaris_core`
> 最後更新：2026-06-25

---

## 一、BQ 實際有哪些文字語料可搜

| BQ 表 | 語意可搜 | `doc_type` / 內容 | 列數 |
|-------|---------|-------------------|------|
| `chunks` | ✅ | `major_news`(5,232)、`transcript`(1,608)、`news`(45) | 6,877 |
| `colpali_pages` | ⚠️ gated | 整頁視覺 PDF（法說簡報）| 5,701 |
| `financial_metrics` | ❌ 只有結構化數字 | EPS / 毛利率 / 營收…等 14 種 metric | 4,620 |
| `events` | ❌ 未接語意搜尋 | `monthly_revenue`(340)、`earnings_call`(254)… | 5,658 |

**結論：語意搜尋目前只覆蓋 chunks（法說逐字稿 + 重大訊息 + 少量新聞）。**
財報全文、法說簡報 PDF、月營收事件文字，目前都無法被問答引擎搜到。

## 二、研究助理（`/ask` / `/research`）

### 2-1. 現有檢索路徑

```
HybridRetriever
  ├─ BM25  ← _FALLBACK_CORPUS（3 筆 hardcoded stub）
  ├─ Vector Search  ← BQ chunks（VECTOR_SEARCH cosine）
  └─ Cohere Rerank（opt-in，需 COHERE_API_KEY）

Deep Research 另加 make_research_search_fn：
  RESEARCH_DOC_TYPE_QUOTAS = {
    "transcript": 5,
    "major_news": 5,
    "news": 3,
  }
```

### 2-2. 缺口清單

| 缺口 | 根本原因 | 影響 |
|------|---------|------|
| **財報文字** | `chunks` 裡無 `doc_type=fin`；財報只以結構化數字存在 `financial_metrics` | 問「台積電財報毛利率分析」→ 無法引用財報原文段落 |
| **法說簡報 PDF** | ColPali 4th 路徑 gated（`colpali_pages` 有料但未接通，待 ≥70% roundtrip 閘） | 問法說視覺圖表、投影片重點 → 無法回答 |
| **月營收公告** | `events.monthly_revenue`（340 筆）未接語意搜尋 | 問「X 月月營收」→ 只能給 KPI 數字，無法引用公告原文 |
| **BM25 stub 極稀** | `_FALLBACK_CORPUS` 只有 3 筆（台積電 2025Q1、2024Q4，鴻海 2025Q1）| 無 Gemini key 時，除台積電/鴻海外其他公司（廣達 3231、聯發科 2454…）BM25 完全找不到資料 |
| **新聞量偏低** | `chunks` 中 `news` 僅 45 筆，quota=3 | 近期新聞覆蓋率低 |
| **估值指標** | BQ `financial_metrics` 無 `pe_ratio`/`pb_ratio` 資料 | 問估值完全無資料 |

## 三、同業比較（`/peer-compare`）

### 3-1. 現有資料來源

```
/peer-compare
  ├─ StructuredStore.list_financials(ticker, period)
  │    → BQ v_financial_metrics_semantic
  │    → 顯示：EPS / 毛利率 / 淨利率 / 月營收YoY / 月營收（KPI 卡）
  │           + 所有共同 metric（financial 表格）
  │
  └─ _search_peer_calls(ticker, period, question)
       → HybridRetriever，filters={doc_type: "transcript"}
       → 只搜法說逐字稿，取最多 8 筆
```

### 3-2. 缺口清單

| 缺口 | 根本原因 | 影響 |
|------|---------|------|
| **同業比較只搜逐字稿** | `_search_peer_calls` filters 寫死 `doc_type=transcript`；`major_news` / `news` 未納 | 重大訊息裡的業務說明、新聞引用都抓不到 |
| **法說簡報未納** | 同上，ColPali 路徑 gated | 法說簡報中的數字、圖表無法引用 |
| **估值欄位空白** | `_VALUATION_METRICS = {"pe_ratio","pb_ratio","ps_ratio"}` 被排除，BQ 本身也無此資料 | 估值頁全是「—」 |
| **法說摘要（AI 生成）缺席** | `summary` 欄位是後端拼接的結構化字串，非 LLM 分析 | 無「這季法說會的核心論述」的文字摘要 |
| **月營收文字脈絡** | 有結構化數字（`financial_metrics`），但 `events.monthly_revenue` 公告文字未接 | 數字有，但無法引用「本月營收創高的原因」這類說明 |
| **新聞 / 輿情** | 前端 `NewsBlock` 標示「待接後端」，API 未實作 | 新聞頁完全空白 |

## 四、跨功能共同缺口

| 項目 | 狀態 | 說明 |
|------|------|------|
| **財報文字化**（PDF → chunks） | 未入庫 | 財報 PDF 需先走 vision OCR → chunks ingestion；目前只有結構化數字 |
| **ColPali 生產啟用** | gated | 需 ≥70% roundtrip 正確率，見 `scripts/colpali_roundtrip_check.py` |
| **BM25 fallback corpus** | 只有 3 筆 | 沒有 Gemini key 時，廣達、聯詠、中信金等其他 17 家公司完全無備援語料 |
| **估值資料**（PE/PB）| 無資料 | BQ 無此表，需接第三方（如 TEJ / 公開資訊觀測站） |
| **月營收事件語意**（events 表） | 未接 | 340 筆月營收公告文字可接語意搜尋，但目前未加入 retriever 流程 |

## 五、可立刻改善的項目（不需新資料）

| 項目 | 改動位置 | 工作量 |
|------|---------|--------|
| 同業比較的 calls 搜尋納入 `major_news` | `api.py _search_peer_calls`，移除 `doc_type` 限制或改 `["transcript","major_news"]` | 小 |
| `RESEARCH_DOC_TYPE_QUOTAS` 加 `earnings_call` | `retriever.py`，確認 BQ 有此 doc_type 後加 quota | 小 |
| 擴充 BM25 fallback corpus | `retriever.py _FALLBACK_CORPUS`，加其他 17 家公司的 stub 條目 | 中（需手寫 stub） |
| 前端同業比較 NewsBlock 接後端 | 需先確認 `/news?ticker=X` 端點存在並有資料 | 中 |
